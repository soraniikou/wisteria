      🪻wisteria

美しい花は　恋の始まりのように狂おしく
愛が深いゆえ　時に激しく執着してしまうこともある
藤の茎のように

愛することや別離を経て　
人は自分自身を愛することを知る
相手がいなければ気づくことができない経験

そんな恋や愛憎から自身を愛する物語をアプリにしました。

体験→https://wisteria310.vercel.app
